## Example User Interaction Flow

1. **User clicks profile button** (top-right red circle)

   - Page navigates to `/profile`
   - Loading spinner shows

2. **API fetches profile data** (GET endpoint)

   - Profile loads into form
   - Spinner disappears

3. **User edits fields**

   - Can upload new profile image
   - Form fields become editable
   - Language selection dropdown available

4. **User clicks "Update Profile" button**

   - Button shows "Saving..." with spinner
   - Inputs become disabled
   - Loading state active

5. **API updates profile** (PUT endpoint)
   - Success: Green alert shows, auto-dismisses
   - Error: Red alert shows with retry button
